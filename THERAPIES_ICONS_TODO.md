## Icon Specifications

- Format: SVG
- Recommended size: 300-350px width, maintaining aspect ratio
- Color: Should work on #d6c68e background (tan/beige)
- Style: Should match the organic, spiritual aesthetic of the design

## How to Export from Figma

1. Open the Figma design file
2. Navigate to each therapy card
3. Select the icon graphic
4. Right-click → "Copy/Paste as" → "Copy as SVG"
5. Save each SVG with the appropriate filename
6. Place in the `public` directory
7. Update `data/therapiesContent.ts` to reference the correct icon paths
